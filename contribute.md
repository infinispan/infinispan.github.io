Infinispan is open source and we welcome contributions from everyone. Whether you're fixing a bug, adding a feature, improving documentation, or helping others in the community — every contribution matters.

### Source Code

The Infinispan source code is hosted on GitHub. Fork the repository, make your changes, and submit a pull request.

<i class="fab fa-github text-orange"></i> [infinispan/infinispan](https://github.com/infinispan/infinispan) — Core project  
<i class="fab fa-github text-orange"></i> [infinispan/infinispan.github.io](https://github.com/infinispan/infinispan.github.io) — Website  
<i class="fab fa-github text-orange"></i> [infinispan/infinispan-simple-tutorials](https://github.com/infinispan/infinispan-simple-tutorials) — Tutorials  

### Report a Bug

Found a bug? Open an issue on GitHub with steps to reproduce it.

<i class="fas fa-bug text-orange"></i> [Submit a bug report](https://github.com/infinispan/infinispan/issues)

### Talk to Us

Join the conversation to ask questions, share ideas, or get help.

<i class="fas fa-comments text-orange"></i> [Zulip Chat](https://infinispan.zulipchat.com) — Real-time chat with the community and core developers  
<i class="fab fa-github text-orange"></i> [GitHub Discussions](https://github.com/infinispan/infinispan/discussions) — Longer-form questions and ideas  
<i class="fab fa-stack-overflow text-orange"></i> [Stack Overflow](https://stackoverflow.com/questions/tagged/infinispan) — Tag your questions with `infinispan`  

### Experiments

Check out our [experiments page](/experiments) to see what we're exploring and prototyping.

### Contributing Guidelines

Before submitting a pull request, check out the [contributing guide](https://infinispan.org/docs/stable/titles/contributing/contributing.html) for coding conventions, build instructions, and the review process.
