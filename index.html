What is Infinispan ?

Infinispan is a distributed in-memory key/value data store with optional schema, available under the Apache License 2.0. It can be used both as an embedded Java library and as a language-independent service accessed remotely over a variety of protocols (HotRod, REST, Memcached and WebSockets). It offers advanced functionality such as transactions, events, querying and distributed processing. Read about our features

Store and retrieve

  • As a Java ConcurrentHashMap
  • With the JCache API
  • With the Functional API
  • Use transactions

Scale and Share

  • Replicate data to all nodes
  • Distribute data to increase capacity
  • Access your data remotely
  • Write your applications in C++
  • Write your applications in C#

Listen and query

  • Detect when data changes in an embedded cache
  • Detect when data changes in a remote cache
  • Index and search your data

Compute

  • Distributed streams
  • Perform complex operations with scripting
  • Integrate with Apache Spark
