Dead trees

Infinispan - the Data Grid Platform by Infinispan founder Manik Surtani and well-known tech writer Francesco Marchioni is now available from Packt Publishing, available in print as well as electronically.

Did you know that some of the Quickstarts are written for the JBoss Data Grid, (JDG) which is Red Hat's officially supported version of Infinispan. The tutorials are just as valid for Infinispan though, so are still worth a look.
