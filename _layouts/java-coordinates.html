Hot Rod Java (Stable)

Maven

Add the following to the dependencies declared in your pom.xml file

    <dependency>
      <groupId>org.infinispan</groupId>
      <artifactId>infinispan-client-hotrod</artifactId>
      <version>{{ site.data.ispn.hotrod.java_stable.version}}</version>
    </dependency>
Gradle

Add the following to the dependencies declared in your build.gradle

    dependencies {
      ...
      implementation 'org.infinispan:infinispan-client-hotrod:{{ site.data.ispn.hotrod.java_stable.version}}'
      ...
    }
Ivy

Add the following to the dependencies declared in your ivy.xml file

    <dependencies>
      ...
      <dependency org="org.infinispan" name="infinispan-client-hotrod" version="{{ site.data.ispn.hotrod.java_stable.version}}"/>
      ...
    </dependencies>
