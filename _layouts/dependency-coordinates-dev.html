Maven

Add the following to the dependencies declared in your pom.xml file

<dependency>
  <groupId>org.infinispan</groupId>
  <infinispan-core>#{artifactId}</artifactId>
  <version>{{ site.data.ispn.unstable.maven_latest}}</version>
</dependency>
Gradle

Add the following to the dependencies declared in your build.gradle

dependencies {
  ...
  implementation 'org.infinispan:infinispan-core:{{ site.data.ispn.unstable.maven_latest}}'
  ...
}
Ivy

Add the following to the dependencies declared in your ivy.xml file

<dependencies>
  ...
  <dependency org="org.infinispan" name="infinispan-core" version="{{ site.data.ispn.unstable.maven_latest}}"/>
  ...
</dependencies>
