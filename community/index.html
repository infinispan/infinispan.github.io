Communications

How to communicate with the Infinispan team

User Forums

Infinispan's user forums should be your first port of call for any user-related issues with Infinispan. Be it a question, wanting to report a bug, looking for advice, or simply sharing some cool stories you may have.

Visit the User Forums

Etiquette

If you are asking for help, have you read through the documentation first? You may find your answer there.

Also, if you are reporting a bug, have you checked our bug tracker first? It may be a known issue.

Chat

The Infinispan community uses Zulip for real-time communications. Join us using either a web-browser or a dedicated application on the Infinispan chat.

Developers Mailing List

This is a mailing list geared towards developing Infinispan itself. It often contains deep, technical discussions on how Infinispan works internally, how new features could be implemented, and is definitely not the place for end user questions such as how to configure, use or even troubleshoot Infinispan. Please use the User Forums for such questions.

If, however, you are helping contribute to Infinispan, we welcome your thoughts, ideas and overall participation on the developers' mailing list.

Join the Developers' Mailing List

Archives

Archives of the Infinispan developers' mailing list can be found on MarkMail which also has an excellent search interface.
