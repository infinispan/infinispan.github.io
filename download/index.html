Download Infinispan

This page contains links to the latest downloads available for the current stable and development releases of Infinispan. Older versions are available in the archive.

Note
Starting with Infinispan 10, we no longer offer a downloadable distribution of the embedded artifacts. Use your favourite dependency management tool (Maven / Gradle / Ivy) using the provided instructions.

Dependency coordinates

Maven

Add the following to the dependencies declared in your pom.xml file 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-core</artifactId>
  <version>10.0.1.Final</version>
</dependency>
...
Gradle

Add the following to the dependencies declared in your build.gradle file 

...
dependencies {
  ...
  implementation 'org.infinispan:infinispan-core:10.0.1.Final'
  ...
}
...
Ivy

Add the following to the dependencies declared in your ivy.xml file 

...
<dependencies>
  ...
  <dependency org="org.infinispan" name="infinispan-core" version="10.0.1.Final"/>
  ...
</dependencies>
...
Docker

Docker Image tag

You can obtain a Docker image for Infinispan Server using the quay.io/infinispan/server:10.0.1.Final tag. Use the following command-line: 
docker pull quay.io/infinispan/server:10.0.1.Final
