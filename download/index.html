Supported versions

Infinispan offers the following support levels:

  • Full support: which includes bug fixes and CVE patches
  • Limited support: which includes select critical bug fixes and CVE patches
  • Limited client support: which includes select critical bug fixes and CVE patches for the Java Hot Rod client only
NOTE: We only support the latest minor release of a major version

The following table describes the currently supported versions with their levels:

Version Support level Java baseline Until
16.x Full support Java 17 (embedded, client), Java 25 (server) 6 months after 17.0 (full), 12 months after 17.0 (limited)
15.2.x Full support Java 17 May 2026 (full), November 2026 (limited)
14.0.x Limited client support Java 11 October 2027
13.0.x Limited client support Java 8 November 2026

Persistent Cache Stores

Cache stores are part of the core Infinispan distribution and therefore are developed and released in-sync with Infinispan itself. Most of these cache stores are included in the infinispan-embedded JAR. Those which require additional dependencies are available in separate artifacts.

Hot Rod Clients

Hot Rod is a wire protocol that Infinispan clients use to talk to a remote grid. It is a binary, platform-independent protocol that was developed in the open as a part of Infinispan.
