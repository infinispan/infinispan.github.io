Dependency Coordinates

Maven

Add the following to the dependencies declared in your pom.xml file

<dependency>
  <groupId>org.infinispan</groupId>
  <infinispan-core>#{artifactId}</artifactId>
  <version>16.1.0.Dev01</version>
</dependency>
Gradle

Add the following to the dependencies declared in your build.gradle

dependencies {
  ...
  implementation 'org.infinispan:infinispan-core:16.1.0.Dev01'
  ...
}
Ivy

Add the following to the dependencies declared in your ivy.xml file

<dependencies>
  ...
  <dependency org="org.infinispan" name="infinispan-core" version="16.1.0.Dev01"/>
  ...
</dependencies>
