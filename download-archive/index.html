Download older versions of Infinispan

This page contains links to the latest downloads available for each major.minor release of Infinispan.

If possible, you should upgrade to a more recent release of Infinispan, as our support policy for older release is quite limited.
Version Release date Release notes Downloads Maven/Gradle/Ivy Container Images
9.3.6
Archived

"Estrella Galicia" 2019/02/13 		Release notes
Fixed issues 		Embedded (SHA-1)
Minimal (SHA-1)
Server (SHA-1)
WildFly modules (SHA-1) 		Dependency coordinates

Dependency coordinates

Maven

Add the following to the dependencies declared in your pom.xml file 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-core</artifactId>
  <version>9.3.6.Final</version>
</dependency>
...
Gradle

Add the following to the dependencies declared in your build.gradle file 

...
dependencies {
  ...
  implementation 'org.infinispan:infinispan-core:9.3.6.Final'
  ...
}
...
Ivy

Add the following to the dependencies declared in your ivy.xml file 

...
<dependencies>
  ...
  <dependency org="org.infinispan" name="infinispan-core" version="9.3.6.Final"/>
  ...
</dependencies>
...
Docker

Docker Image tag

You can obtain a Docker image for Infinispan Server using the jboss/infinispan-server:9.3.6.Final tag. Use the following command-line: 
docker pull jboss/infinispan-server:9.3.6.Final
