Infinispan 8.1.3.Final is out!

April 04, 2016
release

By Sebastian Laskawiec

Dear community,

We are proud to announce the release of Infinispan 8.1.3.Final. https://infinispan.org/download and try it out!

The maintenance release includes:

  • Fixed cluster startup in client server mode

  • Fixed default cache race condition during server start

  • Fixed QueryDSL parser failure with inheritence

  • Improved test suite stability

  • Fixed registering clustered listener (previously it cased SuspectException)

We also performed some upgrades:

  • Protostream was upgraded to 3.0.5.Final

  • XStream was upgraded to 1.4.9

Check out the fixed issues, https://infinispan.org/download/ and tell us all about it on the forum, on our issue tracker or on IRC on the #infinispan channel on Freenode.

Cheers, The Infinispan team

Get it, Use it, Ask us!

We’re hard at work on new features, improvements and fixes, so watch this space for more announcements!

Please, download and test the latest release.

The source code is hosted on GitHub. If you need to report a bug or request a new feature, look for a similar one on our GitHub issues tracker. If you don’t find any, create a new issue.

If you have questions, are experiencing a bug or want advice on using Infinispan, you can use GitHub discussions. We will do our best to answer you as soon as we can.

The Infinispan community uses Zulip for real-time communications. Join us using either a web-browser or a dedicated application on the Infinispan chat.

Infinispan is released under the Apache 2.0 open source license. The source code is available on GitHub.
