Hadoop Connector

Implementation of InputFormat and OutputFormat targeting the Infinispan Server.

GitHub

Version Release date Release notes
0.4 2018/02/05 Release Notes Maven/Gradle/Ivy

Dependency coordinates

Maven

Add the following to the dependencies declared in your pom.xml file 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-hadoop-core</artifactId>
  <version></version>
</dependency>
...
Gradle

Add the following to the dependencies declared in your build.gradle file 

...
dependencies {
  ...
  implementation 'org.infinispan:infinispan-hadoop-core:'
  ...
}
...
Ivy

Add the following to the dependencies declared in your ivy.xml file 

...
<dependencies>
  ...
  <dependency org="org.infinispan" name="infinispan-hadoop-core" version=""/>
  ...
</dependencies>
...
