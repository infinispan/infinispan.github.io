# Project Context: Infinispan Website

The Infinispan website ([infinispan.org](https://infinispan.org/)) is a Jekyll-based static site that serves as the project's public-facing home. It hosts blog posts, documentation links, download pages, tutorials, and release information for the Infinispan ecosystem.

## Shared guidelines
- Use the project's established patterns and terminology. When in doubt, follow the conventions of the existing codebase.
- Prefer clarity over cleverness. This is a long-lived open-source project with many contributors.
- The `develop` branch is the source branch. The `master` branch contains only the compiled site output — never commit source changes to `master`.
