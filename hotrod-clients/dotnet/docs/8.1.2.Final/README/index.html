Documentation

The documentation in this repository is written along the same guidelines as core Infinispan documentation, more of which can be learned about here.

This book is incorporated in the Infinispan Documentation Home

Differences from the core Infinispan documentation

Documentation written for Hot Rod clients should be in a single AsciiDoc document titled index.adoc and stored in this directory. You may use tables of contents and various other AsciiDoc features as described in the document authoring guidelines.

All other aspects of the document authoring guidelines must be adhered to, particularly sections on style and voice.
