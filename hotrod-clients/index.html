Hot Rod Clients

Hot Rod is a wire protocol that Infinispan clients use to talk to a remote grid. It is a binary, platform-independent protocol that was developed in the open as a part of Infinispan.

The Hot Rod specification is available on-line, and several clients exist for this protocol.

Language Version Information Links
Java 10.0.1.Final The reference Hot Rod client, written in Java. This ships with Infinispan. Maven/Gradle/Ivy

Dependency coordinates

Maven

Add the following to the dependencies declared in your pom.xml file 

...
<dependency>
  <groupId>org.infinispan</groupId>
  <artifactId>infinispan-client-hotrod</artifactId>
  <version>10.0.1.Final</version>
</dependency>
...
Gradle

Add the following to the dependencies declared in your build.gradle file 

...
dependencies {
  ...
  implementation 'org.infinispan:infinispan-client-hotrod:10.0.1.Final'
  ...
}
...
Ivy

Add the following to the dependencies declared in your ivy.xml file 

...
<dependencies>
  ...
  <dependency org="org.infinispan" name="infinispan-client-hotrod" version="10.0.1.Final"/>
  ...
</dependencies>
...
