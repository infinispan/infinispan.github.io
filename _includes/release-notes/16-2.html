Infinispan 16.2 UPCOMING

These changes are in development and have not been released yet.

RESP (Redis Protocol)
  • Significant expansion of RESP command coverage: BITFIELD, PSUBSCRIBE/PUNSUBSCRIBE, GEOSEARCH family, COPY, PFCOUNT/PFMERGE (HyperLogLog), JSON improvements
  • Dynamically generated RESP command documentation
  • Aligned command arity, key specs, and ACL categories with Redis
REST API v3 with OpenAPI
  • OpenAPI v3 specification with full coverage of Server, Cluster, Container, Counters, and Cross-site endpoints
  • Embedded Swagger UI
  • Anonymous access to the /server API
  • Protobuf content type support
Core
  • Shared container eviction across caches
  • Fine-grained configuration updates
  • Schema change events
  • State transfer performance and reliability improvements
  • Off-heap support for JDK 23+
  • VarHandle optimizations for MurmurHash3 and multi-byte operations
Server
  • JDK 25 required
  • Heap dumps enabled by default on OOM
  • jdk.incubator.vector module added to startup scripts
  • Improved post-start failure handling
Query & Indexing
  • Migrated from ANTLR 3 to ANTLR 4
  • maxTermCount support for terms aggregations
CLI
  • New bookmark command with credential store integration
  • Upgraded to AEsh 3
  • Enhanced batch failure handling with --on-error support
MCP (Model Context Protocol)
  • ADMIN role enforcement for MCP log resources
  • GC log as an MCP resource
Spring Integration
  • Connection URI support for Spring Boot 4
Cross-site Replication
  • Fixed IRAC entry discard when passivation races with async send
  • Scheduled IRAC re-run after retry to prevent stalled replication
