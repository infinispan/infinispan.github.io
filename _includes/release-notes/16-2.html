Infinispan 16.2 Release Notes

RESP (Redis Protocol)
  • Probabilistic data structures: Bloom Filters, Cuckoo Filters, Count-Min Sketch, Top-K, HyperLogLog
  • New BITOP operations from Redis 8.2
  • GEOSEARCH family of commands
  • COPY, DELEX, DIGEST commands
  • PSUBSCRIBE / PUNSUBSCRIBE pattern-based Pub/Sub
  • BITFIELD / BITOP full implementation
  • AGGREGATE COUNT option for ZUNION, ZINTER, ZUNIONSTORE, ZINTERSTORE
  • Local RESP caches in clustered mode
  • Dynamically generated RESP command documentation
  • Aligned command arity, key specs, and ACL categories with Redis
Server
  • Granular JVM options (JAVA_OPTS_BASE, JAVA_OPTS_NETWORK, JAVA_OPTS_MEMORY, JAVA_OPTS_DEBUG)
  • ECS (Elastic Common Schema) logging support
  • Simplified PEM certificate configuration
  • Minimal boot logging
  • Configurable backpressure for Hot Rod client listeners
  • Backup support for multimaps, RESP, and Memcached caches
  • RocksDB cache store removed from the server distribution
  • Heap dumps enabled by default on OOM
  • Lucene 10.4
Core
  • JSON configuration schemas
  • Pull-based state transfer
  • Memory monitor with configurable thresholds
  • Dynamic eviction based on memory pressure
  • Serialized cache entries on heap
  • Graceful shutdown handling for network partitions
  • Orderly cache scale-down with timeout support
  • Generic Micrometer MeterRegistry support
  • Counter configuration events
  • SIFS persistence store improvements
  • VarHandle optimizations for MurmurHash3
CLI
  • Command suggestions
  • Connection bookmarks with credential store integration
  • Upgraded to AEsh 3
  • Enhanced batch failure handling with --on-error support
Console
  • User permissions page
  • Schema editor with Protobuf syntax highlighting
  • Manageable columns in cache detail view
  • Query history
  • Swagger UI and metrics links
  • Cache deletion from detail view
  • Persistent pagination settings
  • Updated translations (French, Spanish, Italian, Brazilian Portuguese)
MCP (Model Context Protocol)
  • CLI stdio transport for MCP integration
  • GC log as an MCP resource
  • ADMIN role enforcement for MCP log resources
Query & Indexing
  • Migrated from ANTLR 3 to ANTLR 4
  • maxTermCount support for terms aggregations
ProtoStream
  • ASCII string optimizations with VarHandle
  • Kotlin support in annotation processor
  • Copy-on-write serialization context
  • JPMS support with module-info descriptors
Spring Boot
  • Automatic Protobuf schema registration
  • Connection URI support for Spring Boot 4
REST API v3 with OpenAPI
  • OpenAPI v3 specification with full coverage of Server, Cluster, Container, Counters, and Cross-site endpoints
  • Embedded Swagger UI
  • Anonymous access to the /server API
  • Protobuf content type support
Cross-site Replication
  • Fixed IRAC entry discard when passivation races with async send
  • Scheduled IRAC re-run after retry to prevent stalled replication
