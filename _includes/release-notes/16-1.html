Infinispan 16.1 Release Notes

Core
  • Container-based eviction
  • Lucene 10 support (alongside Lucene 9)
Server
  • OpenAPI / Swagger (v3 REST API)
  • Java 25 required for server
  • Lucene 10.3 with SIMD support
  • Pre-start batch scripts
  • AOT cache in the container image
  • Disabled weak/vulnerable authentication mechanisms by default
CLI
  • Batch error handling (fail-fast, fail-at-end, ignore)
