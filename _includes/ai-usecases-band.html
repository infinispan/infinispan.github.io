AI Use Cases

Semantic Caching

Cache LLM responses by semantic similarity. When a new prompt is similar in meaning to a cached one, return the cached response instantly — saving tokens, reducing latency, and cutting API costs by up to 70%.

Vector Store for RAG

Store document embeddings and retrieve relevant chunks via kNN search to ground LLM responses in your data. Infinispan's distributed vector indexes scale across your cluster with sub-millisecond retrieval.

Agentic AI Caching

Cache tool results, API responses, and intermediate computations for AI agents. Multiple agents share the same distributed cache, so one agent's lookup benefits all others.

Conversation Memory

Persist multi-turn conversation history across application restarts and instances. TTL-based expiration cleans up old sessions automatically. Cross-site replication for global apps.
